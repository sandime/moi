# Sherri Cruz — Author & Journalist Landing Page

A personal landing page with an **atomic / retro Googie** design aesthetic.

## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `sherri-cruz` or `sherricruz.github.io`)
2. Upload `index.html` to the root of the repository
3. Go to **Settings → Pages**
4. Under **Source**, select `Deploy from a branch` → choose `main` and `/ (root)`
5. Click **Save** — your site will be live at `https://yourusername.github.io/repo-name`

## ✏️ Customizing

| Section | What to edit |
|---|---|
| **Author photo** | Replace the `📸 Author photo coming soon` placeholder with an `<img>` tag |
| **Flash Fiction** | Replace the coming-soon box in `#flash` with story cards like the ones in `#stories` |
| **Doodles** | Replace the coming-soon box in `#doodles` with `<img>` tags in a grid |
| **Colors** | Edit the CSS variables at the top of the `<style>` block |

## 🎨 Design Notes

- Palette: atomic orange `#FF5C1A`, starburst yellow `#FFD21E`, teal `#00C5B5`, coral `#FF3E6C`
- Fonts: Orbitron (display), Exo 2 (body/italic), Space Mono (mono)
- Inspired by 1950s–60s Googie architecture and atomic-age graphic design
